==================================
API Мобильная торговля "ОПТ-СЕРВИС"!
==================================

Протокол взаимодействия с сервером “ОПТ-СЕРВИС”.

Данное руководство описывает протокол обмена между Учетной системой заказчика с Сервером “ОПТ-СЕРВИС” (далее - Сервер).

Используя статьи данного документа IT-специалисты могут выполнить самостоятельное внедрение у себя на площадках
 
Для реализации обмена используются **POST запросы** в соответствующих форматах, обмен осуществляется в файлах формата XML.

В заголовке (header) обязательно должен присутствывать параметр “key” - идентификатор УС.

"key" - выдается отделом поддержки и идентифицирует заказчика и УС, данный мой ключ можно сгенерировать и зарегистрировать самостоятельно (см. документацию нижe).

Для инициализации "key" свяжитесь со службой сопровождения по e-mail support@opt-service.com
